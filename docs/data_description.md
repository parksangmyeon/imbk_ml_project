# 데이터 설명



## 사용 데이터



본 프로젝트는 iM뱅크 교육용 법인 고객 금융거래 데이터를 활용했습니다.  

원본 고객 데이터와 금리현황 데이터는 보안 및 저작권 문제로 GitHub 저장소에 포함하지 않았습니다.



## 분석 목적



대구·경북 우수 법인 고객의 금융 거래 패턴을 기준으로 타지역 내 유사한 기업금융 후보지를 탐색하기 위해 분석을 수행했습니다.



## 분석 단위



- 법인 고객 단위 금융거래 데이터

- 지역, 업종, 수신, 여신, 디지털 거래, 외환, 카드 관련 변수 포함





## 주요 파생 변수

| 변수명 | 설명 |
|--------|------|
| 총예금잔액 | 요구불예금, 거치식예금, 적립식예금 등 예금 잔액 합계 |
| 총대출잔액 | 운전자금대출잔액과 시설자금대출잔액 합계 |
| 총디지털거래액 | 인터넷뱅킹, 스마트뱅킹, 폰뱅킹 거래액 합계 |
| 총오프라인거래액 | 창구거래 및 ATM 거래 등 오프라인 채널 거래액 |
| 총외화실적 | 수출입 등 외화 거래 실적 |
| 총카드소비 | 신용카드 및 체크카드 사용금액 |
| 총예금좌수 | 고객과 은행 간 거래 관계 깊이를 나타내는 변수 |
| 예대마진 | 대출이자 - 예금이자 |



## 예대마진 산출 방식



예대마진 산출에는 고객별 예금·대출 잔액과 2022\~2024년 월별 금리 데이터를 활용했습니다.



- 예금이자 = 총예금잔액 × 평균예금금리

- 대출이자 = 총대출잔액 × 평균대출금리

- 예대마진 = 대출이자 - 예금이자



## 전처리 요약



- 지역 미상 데이터 정리

- 대구·경북 지역을 통합

- 예대마진 결측 및 이상치 처리

- 금융 금액 변수 `log1p` 변환

- StandardScaler 기반 표준화

- 대구·경북 데이터와 타지역 데이터 분리

- 대구·경북 우수집단과 타지역 데이터 결합 후 2차 클러스터링 수행



## 데이터 비공개 안내



본 프로젝트의 원본 금융 데이터와 금리현황 데이터는 공개하지 않습니다.  

따라서 저장소 내 노트북은 원본 데이터 없이 완전 재실행할 수 없으며, 주요 실행 결과와 분석 흐름은 README와 노트북 출력으로 확인할 수 있습니다.
